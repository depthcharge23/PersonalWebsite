Projects

Kanto Pokedex App

The Kanto Pokedex project was a personal project I wanted to work on. The idea was to scrape Pokemon data from the website, Bulbapedia, and display it in a pokedex format.

The front-end section of the app was created in React with the help of Bootstrap. The back-end was created in NodeJS using the ExpressJS server framework. All of the data was scraped off of Bulbapedia using Python and the Beautiful Soup library. The data was loaded into a MongoDB instance. The app was hosted on Microsoft Azure.
